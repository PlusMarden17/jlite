# JLite

JLite це простий кодовий редактор для Java

Наразі у JLite є:
1. Створення/Відкриття/Збереження файлів
2. Попередження
У майбутньому буде додана підсвітка синтаксису, відкриття кілька файлів підряд та багато іншого
